## Limitations

- Static scanning cannot prove a symbol is visible from the future `exec` runtime environment.
- Static scanning cannot prove object instance paths. For example, `ModStoreClass:NewMod` may be reached through a runtime object rather than the class name itself.
- Static scanning cannot prove side effects or mutation safety.
- Local tables may be module-private unless returned or assigned into a global/module surface.
- Large data tables must be filtered at runtime and should not be returned wholesale.
